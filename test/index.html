Network Data

This project is a suite of methods and types to aid in working with data that is interconnected in a network. The following expresses examples that show how to work with the data and see some visualizations of the results and operations the methods perform.

Making the Data

makeNetwork

This method helps convert an existing data structure into a NetworkData object which is the basis for operations in this library.
// Start with some data const nodes = genNodes(100); const edges = genEdges(nodes, 1000); // Convert the data to NetworkData const network = await makeNetwork({ edgeData: edges, nodeData: nodes, nodeId: row => row.UID || "", edgeId: row => row.UID || "", nodeMeta: row => ({ name: row.name }), edgeMeta: row => ({ name: row.name }), edgeA: row => row.UID_A || "", edgeB: row => row.UID_B || "", edgeValues: row => ({ ab: row.numMetric, ba: row.numMetric }), nodeValues: row => row.numMetric });
